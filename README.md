# Js-Csharp-UploadDownload
javascript+Csharp:Upload File and Download File  
　　
　　
　　
　　
　
### Web
　　common 引用的文件  
　　config.js 配置文件  
　　服务器IP及端口(下载的文件所在文件夹需要发布才能访问)，以及连接C#的路径  
　　download.js 下载文件  
　　upload.js 上传文件  
　　worker.js 开辟双线程(浏览器)　　
　　
### WebApplication
　　C#工程  
　　/WebApplication/bin/Config.xml 配置项目所在路径及下载文件的路径
